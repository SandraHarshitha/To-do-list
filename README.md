# 📝 Python To-Do List Application

A simple command-line To-Do List application built with Python. This project allows users to manage their daily tasks by adding, viewing, and removing tasks through an interactive menu-driven interface.

## Features

* View all tasks
* Add new tasks
* Remove existing tasks
* User-friendly menu interface
* Basic error handling for invalid inputs

## Technologies Used

* Python 3

## How It Works

The application provides the following options:

1. **View Tasks** – Displays all current tasks.
2. **Add Task** – Adds a new task to the to-do list.
3. **Remove Task** – Removes a task by selecting its number.
4. **Exit** – Closes the application.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/todo-list-python.git
```

2. Navigate to the project directory:

```bash
cd todo-list-python
```

3. Run the program:

```bash
python todo.py
```
